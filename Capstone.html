Automated Parking Management System

Capstone Design

About

In this project, we designed an automated parking management prototype system using an Arduino in order to self preserve a prototype parking lot. This project included a software an a hardware portion which can be seen below. First a vehicle enters the vicinity of the parking lot in which a customer who booked (through our webapp) will be asked to present a QR code (which they get after booking an appointment), the camera (esp32 cam) will register this QR code which will then turn on the servo motors (the gates) to allow the vehicle to enter. Once a vehicle finds a suitable parking spot, the weight sensors will then detect that vehicle and update the database

This project was divided into a hardware and software component.
