Keylogger Project

Python-based Keyboard Event Logger

Project Overview

This Keylogger Tool is my very first Python project — built from scratch as I dive into coding for the first time. It captures keyboard input events in real-time, logging keystrokes to help me understand how low-level event handling and data capturing works.

As a beginner, this project was both a challenge and an exciting learning experience, giving me practical insight into programming fundamentals and Python’s capabilities.

While the tool has basic features, it lays the foundation for more complex software I plan to build as I continue improving my skills.

Important: This project is meant for educational purposes only. Always use such tools responsibly and ethically — never on unauthorized systems.

I’m proud of this milestone on my coding journey and excited for what’s next!

Key Features

Real-time Keystroke Capture

Records keystrokes in real-time with minimal latency.

Secure Log Storage

Logs are stored locally in encrypted files for privacy and security.

Configurable Options

Adjust logging intervals, target applications, and output formats.

Tech Stack

This project was developed using Python 3, focusing on core libraries like pynput for capturing keyboard events, logging for secure file management, and basic encryption methods for log privacy.

I leveraged beginner-friendly Python tools and modules to create a functional and extensible codebase that is easy to understand and maintain.

How to Use

1. Clone or download the repository from GitHub.
2. Install the required dependencies using pip install -r requirements.txt.
3. Run the script using python keylogger.py.
4. The program runs silently, capturing keystrokes and saving them securely to a local encrypted log file.
5. To stop logging, safely terminate the script via keyboard interrupt (Ctrl + C).

Please use responsibly and only on devices you own or have explicit permission to test on.
