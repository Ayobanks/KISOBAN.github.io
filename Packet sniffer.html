Packet Sniffer tool

Capstone Design

About

In this project, I built a basic packet sniffer using Python to understand how data moves across a network. The tool captures network packets in real time and displays key information like source and destination IP addresses, the protocol used (e.g., TCP, UDP), and the raw data being sent. This helped me learn how to monitor and analyze traffic, a core skill for roles like SOC Analyst and Security Engineer.

How It Works

The tool captures network packets in real-time using Pythonâ€™s socket library. It decodes packet headers to extract key info such as source and destination IPs and protocols, then displays this data for analysis.

Tech Stack

  • Python
  • Socket library
  • Wireshark (for packet analysis reference)

Usage

Run the script with administrator/root privileges. Use the command: python packet_sniffer.py. Make sure Python is installed on your machine.

Skills Gained

  • Network packet analysis
  • Python scripting and socket programming
  • Understanding TCP/IP protocols
  • Data parsing and debugging

Challenges

Handling raw packet data and ensuring the sniffer runs with the correct permissions were challenging. I overcame these by carefully parsing protocol headers and running the script with admin rights.
