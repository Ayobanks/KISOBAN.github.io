Card Detection Program

Python Image Processing

About

In this project, I designed a YuGiOh Card detection program in python utilizing the openCV library. This involved the use of many different filters in order to extract key information of the image and then processing them using general python commands. The Diagram on the left shows the simple representation of a YuGiOh card. A YuGiOh Card has an attribute, a level and a card-type which can all be seen on the diagram By using these 3 key parts of the card, we can extract all the information we need to know about the card and use it to determine what kind of card is presented to the program.
